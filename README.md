### 功能描述
**本版本的友情链接可以支持以下的功能：**

 - 1、方便地在侧边栏添加友情链接。
 - 2、支持两种输出方式。一种为函数输出方式，主要用于侧边栏的友情链接，或者模板开发者设计的友情链接模板等。另一种方式为HTML标签式输出，主要方便用户建立自己的友情链接页面。
 - 3、支持文字友链、图片友链、图文混合友链等。内设这三种默认的输出方式，支持自定议设定输出规则。
 - 4、支持友链分类，方便管理。
 - 5、支持友链启用禁用，方便管理。
 - 6、支持友链拖拽排序，方便管理。
 - 7、支持友链邮箱解析头像链接，方便用户添加无图片的友链。
 - 8、支持增加自定义字段，方便用户做一些个性扩展。
